Finding IP

  • Wireless IP Address: ipconfig getifaddr en1
  • Ethernet IP Address: ipconfig getifaddr en0
  • More straightforward way: ifconfig | grep "inet " | grep -Fv 127.0.0.1 | awk '{print $2}'
